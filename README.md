# camera-viewer

A simple, fullscreen webcam viewer hosted with GitHub Pages.

## Live Site

[https://joshua-8.github.io/camera-viewer/](https://joshua-8.github.io/camera-viewer/)

## Usage

1. Open the site in a modern browser.
2. Click **Start Camera**.
3. Grant camera permissions when prompted.
4. The webcam stream is requested at **1920×1080** resolution (the browser will use the closest resolution your camera supports).
5. Use the **Fullscreen** button (bottom-right) to enter or exit fullscreen mode.

## GitHub Pages Setup

The site is served directly from the root of the `main` branch.  
In repository **Settings → Pages**, set the source to:

- **Branch:** `main`
- **Folder:** `/ (root)`
